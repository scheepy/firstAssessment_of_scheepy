命令 : subEx
說明：訂閱指定商品，呼叫此命令之後，會返回該商品所有profile的版本號，並在之後，單一profile更新版本，將會收到通知。
條件：Json字串
 Key  類型  說明
 s  String  Service : APConfig
 c  String  Command : subEx
 d  Json  Data
Key  說明
 p(product)  Json字串，Key為profile，value為版本號。
 u(user)  Json字串組成的array，內容為company、product、user、profile
 msg  當cd不為0時的錯誤說明
 product  String  欲訂閱的商品。
例：{"s":"APConfig","c":"subEx ","d":{"product":"aaa"}},"r":"anyString"}

回傳格式
 Key  說明
 s  APConfig
 c  subEx
 d  紀錄所有的profile版本
Key  說明
 p(product)  Json字串，Key為profile，value為版本號。
 u(user)  Json字串組成的array，內容為company、product、user、profile
 msg  當cd不為0時的錯誤說明
 cd  錯誤代碼
 ErrorCode  說明
 0  成功
 r  使用者帶入的r
例：{"s":"APConfig","c":"search","d":{"cpuf":{"company":"as"},"data":[{"company":"as","profile":"default"},{"company":"as","product":"yc-master","profile":"default"},{"company":"as","product":"yc-master","user":"user001","profile":"default"},{...}]},"r":"123"}

命令： search
說明：用來查詢符合條件的使用者名稱與相關的所有設定
條件：Json字串
 Key  類型  說明
 product  String  任意字串。預設為空值，可代入"*"來代表任意值。
 user  String  任意字串。預設為空值，可代入"*"來代表任意值。
 profile  String  任意字串。預設為空值，可代入"*"來代表任意值。
例：{"s":"APConfig","c":"search ","d":{"profile":"anyString","data":{ (略) }},"r":"123"}

回傳格式
 Key  說明
 s  APConfig
 c  search
 d  資料Josn
Key  說明
 cpuf  Json字串，使用者代上來的company(c)、product(p)、user(u)、profile(f)
 data  Json字串組成的array，內容為company、product、user、profile
 msg  當cd不為0時的錯誤說明
 cd  錯誤代碼
 ErrorCode  說明
 0  成功
 r  使用者帶入的r
例：{"s":"APConfig","c":"search","d":{"cpuf":{"company":"as"},"data":[{"company":"as","profile":"default"},{"company":"as","product":"yc-master","profile":"default"},{"company":"as","product":"yc-master","user":"user001","profile":"default"},{...}]},"r":"123"}
